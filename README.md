# 📝 To-Do App (for (;; ) 팀 미션)

React를 활용해 제작한 간단하고 직관적인 **To-Do 애플리케이션**입니다.  
사용자가 할 일을 입력하고, 완료 여부를 체크하며, 필요 없어진 항목을 삭제할 수 있는  
기본 기능 중심의 실습 프로젝트입니다.

본 프로젝트는 **for (;;)** 팀이 협업으로 진행한 팀미션입니다.

---

## 🚀 프로젝트 소개

이 프로젝트의 목적은 아래 세 가지에 집중하고 있습니다:

1. **React 컴포넌트 구조 이해**
2. **상태 관리(useState) 기반 기능 구현**
3. **UI/UX 개선과 협업 경험 습득**

직접 컴포넌트를 분리하고, 이벤트 핸들링을 적용해보며  
실무에서의 기본적인 프론트엔드 개발 흐름을 경험할 수 있었습니다.

---

## 🎯 팀미션 목표

- 할 일 관리 기능을 중심으로 웹 서비스의 기본 구조를 이해한다.
- React를 활용한 **컴포넌트 분리**, **상태 관리**, **UI 구성**을 경험한다.
- 팀원 간 역할 분담 및 협업을 통해 코드 작성과 리뷰 과정에 익숙해진다.
- 사용자에게 직관적인 **할 일 관리 경험**을 제공하는 것을 목표로 한다.

---

## ✨ 주요 기능 (Features)

### ✔ 1. 할 일 추가 (Add Task)
- 입력창에 텍스트를 입력하면 리스트에 새 항목이 추가됩니다.
- 공백 입력 방지 기능 포함.

### ✔ 2. 완료 체크 (Toggle Complete)
- 체크박스를 클릭하면 완료/미완료 상태가 전환됩니다.
- 완료 항목은 스타일로 시각적 구분.

### ✔ 3. 할 일 삭제 (Delete Task)
- 각 항목의 삭제 버튼을 눌러 특정 항목을 제거할 수 있습니다.

### ✔ 4. UI/UX 개선 요소
- 줄 간격, 정렬, 가독성 중심 UI 구성
- 간단한 반응형 대응
- 첫 번째 항목 구분선 등 레이아웃 및 시각적 개선
